## 🚀 Features

- 🎯 **Radial Color Wheel** — intuitive hue & saturation selection  
- 🔁 **Live Sync System** — HEX, RGB, and HSL always stay in sync  
- 🌗 **Lightness Control** — fine-tune brightness with a slider  
- 🎨 **Color Harmonies** — complementary, triadic, and analog palettes  
- 🕘 **History System** — saves recently used colors (localStorage)  
- 📋 **One-Click Copy** — instantly copy HEX values  
- ⚡ **Zero Dependencies** — pure vanilla JavaScript

---

## 🧠 How It Works

Chroma Wheel uses a canvas-based radial color system:

- **Angle → Hue**
- **Radius → Saturation**
- **Slider → Lightness**

Everything else (RGB, HEX, HSL, harmonies) is derived in real time from this core model.
